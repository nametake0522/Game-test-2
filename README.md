# Vibe Coding Game 🎮

シンプルなブラウザゲーム。敵を避けながらアイテムを集めよう！

## 🎯 ゲームの遊び方

- **矢印キー（↑↓←→）** でプレイヤー（水色の四角）を操作
- **黄色いアイテム** を集めてスコアを稼ぐ
- **赤い敵と攻撃** を避ける
- 敵や攻撃に当たるとゲームオーバー

## 🎮 ゲームの特徴

- キーボード操作でリアルタイムに動作
- 敵がプレイヤーを追跡してくる
- 敵が周期的に攻撃を発射
- スコア機能でゲーム性を追加
- シンプルで分かりやすいUI
